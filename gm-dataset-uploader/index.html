SOP & Rules

Standard Operating Procedures for Data Collection

Composite Items

Pringles Can 502 Trash

If intact (Tube + Metal + Cap), label as Trash. Only label parts if separated.

Cigarette Box 502 Trash

If intact (Foil + Plastic + Box), label as Trash. Tear apart to label Cardboard (301).

Cans & Labels

Tin Can + Label 202 Metal

If paper label is glued on, treat WHOLE object as Metal. Do not label paper separately.

Aerosol Can 202 Metal

Empty spray cans go to Metal. Cap usually goes to Plastic (103).

Local Items (Malaysia)

Ikat Tepi 502 Trash

Drink bags with string/straw are contamination. Do not label as soft plastic.

Tetra Pak 303 Carton

Milo/Yeo's drink boxes are NOT cardboard. Use specific category 303.

Common Mistakes

Dirty/Greasy 501 Dirty

Pizza boxes with oil, or unwashed plastic containers with curry.

Receipts 502 Trash

Thermal paper cannot be recycled. Do not label as Paper.
